Tentang Saya

Halo! Saya Alexandra

Saya seorang Full Stack Developer dengan pengalaman lebih dari 5 tahun dalam membangun aplikasi web modern. Saya memiliki passion untuk menciptakan solusi digital yang tidak hanya indah tetapi juga fungsional dan mudah digunakan.

Spesialisasi saya mencakup pengembangan frontend dengan React.js dan Vue.js, serta backend dengan Node.js dan Express. Saya selalu mengikuti perkembangan teknologi terbaru untuk memberikan solusi terbaik bagi klien.

Di waktu luang, saya menikmati hiking, fotografi, dan berkontribusi pada proyek open source. Saya percaya bahwa kolaborasi dan pembelajaran berkelanjutan adalah kunci kesuksesan dalam industri teknologi.
